# Task 1 - Full Stack Development Project

## 📌 Project Overview
This is a beginner full stack project created as part of internship tasks.

The project includes a user registration form where users can enter their name and email. After submission, the entered data is displayed on a success page.

## 🚀 Features

- User Registration Form
- Node.js + Express Server
- EJS Templating
- Form Submission Handling
- Dynamic Success Page
- Stylish Responsive UI

## 🛠 Technologies Used

- HTML
- CSS
- Node.js
- Express.js
- EJS

## ▶️ How to Run

1. Download project files
2. Open terminal in project folder
3. Run:

npm install

4. Then run:

node app.js

5. Open browser:

http://localhost:3000
